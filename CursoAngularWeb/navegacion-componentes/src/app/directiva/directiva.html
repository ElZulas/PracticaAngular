Directivas en angular💻

Las directivas en Angular son instrucciones que se aplican al DOM (Document Object Model) y le indican cómo comportarse o qué hacer. Las directivas permiten agregar, modificar o eliminar elementos del DOM y manipular la forma en que se muestra la interfaz de usuario.
Existen dos tipos principales de directivas en Angular:

Directivas incorporadas (Built-in Directives): Angular viene con un conjunto de directivas incorporadas que están listas para usar. Estas directivas se proporcionan como parte del marco de trabajo y cubren casos de uso comunes.

Directivas personalizadas (Custom Directives): Además de las directivas incorporadas, Angular permite a los desarrolladores crear sus propias directivas personalizadas para satisfacer necesidades específicas de la aplicación. Estas directivas personalizadas se pueden utilizar de manera similar a las directivas incorporadas y proporcionan una manera de extender el comportamiento del DOM.
Angular proporciona una serie de directivas incorporadas que son muy útiles para la manipulación del DOM y la gestión de la interacción con el usuario. Algunos ejemplos de directivas incorporadas son ngIf, ngFor, ngSwitch, ngStyle y ngClass.

Las directivas personalizadas se crean mediante el uso de decoradores especiales en clases TypeScript y se aplican a elementos HTML utilizando un selector personalizado.

En resumen, los componentes y directivas son elementos esenciales en Angular que permiten la construcción de aplicaciones web modulares, reutilizables y fáciles de mantener. Los componentes representan partes de la interfaz de usuario con su propia lógica y vista, mientras que las directivas permiten manipular el DOM para modificar la apariencia o el comportamiento de los elementos. Juntos, estos dos conceptos forman la base para el desarrollo de aplicaciones sólidas en Angular.
